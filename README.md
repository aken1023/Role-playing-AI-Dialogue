# Role-playing-AI-Dialogue

一個角色扮演AI對話系統，提供多種角色選擇和語音回應功能。

## 功能特色
- 多種角色選擇
- 語音合成回應
- 響應式設計
- 對話歷史記錄

## 安裝步驟
1. 克隆本倉庫
```bash
git clone https://github.com/aken1023/Role-playing-AI-Dialogue.git
```
2. 安裝依賴
```bash
pip install -r requirements.txt
```
3. 運行應用
```bash
python app.py
```

## 使用說明
1. 選擇角色
2. 輸入訊息並發送
3. 點擊喇叭圖標可重聽AI回應
